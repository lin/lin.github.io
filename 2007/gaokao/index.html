    高考具体的情况

    我总估分平均为670分，实际考分是666分，主要是数学高估了5分

    语文，125分，估分123分

    1. 前30道阅读满分
    2. 翻译扣一分，诗词鉴赏扣3分
    3. 古诗词，闻道龙标过五溪 写错，花之隐逸者也 写错，扣两分
    4. 大阅读，扣三分
    5. 19题扣一分，20题扣4分
    6. 作文，实际扣了约9分，估分时，估计扣8-10分

    数学，135分，估分140分

    1. 提前四十分钟答完
    2. 最后的导数，往年都是数列压轴，这个题很新颖，但我还是五分钟就做上了，这道题应该是满分
    3. 倒数第二题，数列是我的强项，我用了一个自己的方法，不是标准答案，我以为最多扣3分，但是实际一分没给，扣了8分，也就高估了5分
    4. 选择题全对，可能是高考数学最简单的考题之一，我大约五分钟就做了前九道，第12题，我用特值法，毫不费力，全卷打完之后，我才严密的证明
    5. 填空题全对，但是14题，我们老师说正态分布超纲不考，我的很多同学在这个题目上失分
    6. 第18题的概率统计题，我第二问看错了数，上一问的0.2，我抄写成了0.02，导致我所有考试第一次概率统计题失分，丢掉了6分，但我平时考试总会有5、6分的马虎，在可以接受的范围内

    英语，141分，估分141分

    1. 单选错一题，1分，32题，过去时语法错误
    2. 完形错一题，1.5分，48题，in flight
    3. 阅读全对
    4. 改错错一题，1分，82题，Each of
    5. 上面客观题，一共失分3.5分
    6. 作文，失分5分，因为考题简单，据说作文扣分很严格，所以我算是幸运，分数很高

    理综，265分，估分267分

    生物，失分8分，64分

    1. 选择全对
    2. 30题，第一部分满分
    3. 30题，第二部分，第一问和第二问失分，扣8分
    4. 31题，满分

    化学，最短板学科，只错两分，106分

    1. 选择全对
    2. 26题，满分
    3. 27题，推断，满分
    4. 28题，第三小问，失两分
    5. 29题，有机，满分

    物理，最擅长学科，翻车最严重，失分24分，98分

    1. 17题选择，没有选择D选项，扣三分
    2. 20题选择，没有选择D选项，扣三分
    3. 22题，全对
    4. 23题，全对
    5. 24题，扣了11分，老师说，非弹性碰撞不考，我也就现场推导这个式子，有些紧张，并且花费了很久，但是因为往常不写中间的结论，我没有把式子写下来，最后的结果还是算错的，这是我平时考试，唯一一次物理大题第二题出错，而且老师还说，这个题省里开会，中间的8分最后给分，也让我估分不保守的情况下多了8分
    6. 25题，扣了8分（第一问两分，第二问六分），我记得自己推出来一个悖论，所以这个题耽误了我很多时间
