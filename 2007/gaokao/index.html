    高考具体的情况

    我总估分平均为670分，实际考分是666分，主要是数学高估了5分

    语文，125分，估分123分

    1. 前30道阅读满分
    2. 翻译扣一分，诗词鉴赏扣3分
    3. 古诗词，闻道龙标过五溪 写错，花之隐逸者也 写错，扣两分
    4. 大阅读，扣三分
    5. 19题扣一分，20题扣4分
    6. 作文，实际扣了约9分，估分时，估计扣8-10分

    数学，135分，估分140分

    1. 提前四十分钟答完
    2. 最后的导数，往年都是数列压轴，这个题很新颖，但我还是五分钟就做上了，这道题应该是满分
    3. 倒数第二题，数列是我的强项，我用了一个自己的方法，不是标准答案，我以为最多扣3分，但是实际一分没给，扣了8分，也就高估了5分
    4. 选择题全对，可能是高考数学最简单的考题之一，我大约五分钟就做了前九道，第12题，我用特值法，毫不费力，全卷打完之后，我才严密的证明
    5. 填空题全对，但是14题，我们老师说正态分布超纲不考，我的很多同学在这个题目上失分
    6. 第18题的概率统计题，我第二问看错了数，上一问的0.2，我抄写成了0.02，导致我所有考试第一次概率统计题失分，丢掉了6分，但我平时考试总会有5、6分的马虎，在可以接受的范围内

    英语，141分，估分141分

    1. 单选错一题，1分，32题，过去时语法错误
    2. 完形错一题，1.5分，48题，in flight
    3. 阅读全对
    4. 改错错一题，1分，82题，Each of
    5. 上面客观题，一共失分3.5分
    6. 作文，失分5分，因为考题简单，据说作文扣分很严格，所以我算是幸运，分数很高

    理综，265分，估分267分

    2. 30题，第一部分满分
    3. 30题，第二部分，第一问和第二问失分，扣8分
    4. 31题，满分

    化学，最短板学科，只错两分，106分

    1. 选择全对
    2. 26题，满分
    3. 27题，推断，满分

    5. 29题，有机，满分

    物理，最擅长学科，翻车最严重，失分24分，98分

    1. 17题选择，没有选择D选项，扣三分
    2. 20题选择，没有选择D选项，扣三分
    3. 22题，全对
    4. 23题，全对
    5. 24题，扣了11分
    6. 25题，高考题目有错，扣了8分（第一问两分，第二问六分），我记得自己推出来一个悖论，所以这个题耽误了我很多时间

    物理第24题的展开

    1. 我们的物理老师说，这样的弹性碰撞不考，对于很多其他名校学生，很基础的二级结论，我以及我的同学们必须现场推导这个式子
    2. 这个题目的推导有这很大的技巧性，如果是直接消元，而不是平方差公式消元，计算量很大，陷阱也很多，会限于泥潭
    3. 这道题从物理学的角度来说，能够识别模型，列出初始方程，物理能力的考察就已经结束了，这也就是，往常不写中间的结论，我没有把式子写下来，最后的结果还是算错的
    4. 而且后来绥化一中的老师还说，这个题省里开会，中间的8分最后给分，也让我估分不保守的情况下多了8分
    5. 在高考的环境下很紧张，并且为了不出错，要花费很久，我第一次推导，通过消元的方式，发现太麻烦，才想到平方差的约分技巧（类似于数学的点差法），但这时候时间已经过去了很久
    6. 第二道大题的难度突然变成了第三道大题的难度，与此同时，因为，这是我平时考试，唯一一次物理大题第二题出错
    7. 这个题目，其实有双重的责任，一是老师不应该说不考查，第二，命题者极不负责，不应该突然给中间解出的结论八分的赋值，至少应该作为一个需要求解的量，明确的让学生列出
    8. 这个题目，明显是对名校生，对竞赛生，有利的一个题目，是对他们额外练习，老师提供的更大知识库，更好的答题能力的一个体现
    9. 作为第二个大题，对于大多数好学生，应该是必拿分的一道题，对于心态的打击也是很大的

    物理第25题出错

    1. 题干的描述有矛盾，圆心必然是在第三象限，而不是在答案所说的第二象限，这就使得必然是y轴正向成锐角，刘翔和我说，他也发现了这个错误
    2. 我的磁场大小求出来是对的，但没有写中间变量 R 的表达式，扣掉了六分，我觉得很扯淡
    3. 我当时是用数学解析几何的办法，解出来的半径表达式，而不是标准答案的那种方法，具体的，也很简单，把 AC和OC的方程联立，就能得到 O 的坐标，$x_o = -\frac{h^2}{l}$，$y_o = -\frac{h^2 + l^2}{2h}$，这时候，只要算出来OC的长度，就是 $R$ 了，简单粗暴，但这么算，就会意识到，题目是错的，就会让人崩溃，觉得自己是不是 O 的坐标求错了，陷入一种恐慌和自我怀疑。
    4. 那其实这就有一个随机事件，如果一个学生用了这个方法，就会意识到考题有误，那就看当时的运气了，我就是运气不好

    实力

    1. 我数学150满分是很容易做到的，但是考虑马虎，就算是144分是很常规的
    2. 物理的第二道大题，我几乎从来不错的，所以这11分是因为老师的误判造成的，我的同学们还有一个正态分布的误判
    3. 第三道大题，因为第二道大题，挤占了很多的时间，我理综285，并不是一个很难的分数
