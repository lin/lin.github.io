    对购置房产的分析（2018年08月30日）

    这是我妈追着让我买婚房时的分析

    购置房产利弊分析

    1，买房子的弊端

    1.1 经济发展属于泡沫和债务高点，高价入，折损大

    1.2 消耗资金储备，对风险和危机的承受能力减弱，不能有大病大灾

    1.3 减少了资金流动性，不具备投资能力，压抑生活品质

    2，不买房子的弊端

    2.1 房价继续爆涨，当刚需时没有能力购入

    反驳：各种迹象表明，现在的经济处于瓶颈期，很难再次发生过去三十年的高速发展，房价已经畸形，政府不会不管制（所有人都认为房子是炒高的，而且是高杠杆抬高的）

    2.2 孩子上学没有学区房优势

    反驳：十年之后的变化比较大，政策，财富量，房价，生活地点等等。这件事不算“人无远虑，必有近忧”。

    2.3 压抑了住大房子的享受

    反驳：年轻的时候要先牺牲，年老的时候才能更多的享受

    下面是我 2018年写在推特上 的几个博文

    1. 要想真正的理解这个世界，最重要的一个假设就是，社会主体的组成人员是很盲从的，缺乏独立思考的能力，这是基本出发点，知道了这个对于理解很多的社会现象就清晰了很多（Aug 3, 2018 ）
    2. 结婚就要贷款买房，在我眼里就像男人要留长辫子一样愚昧，一方面消耗了人生探索的财务基础，另一方面牵制了探索的自由。买房子夸张一点就是买空气，一辈子就为了能够呼吸而活，太糟心了（Aug 13, 2018）
    3. 如果信贷不是用来购买生产材料和提高生产率，而是用来享乐和提前消费，信贷就是穷人活该穷的原因，好比卖身赎命（Sep 1, 2018）
    4. 用来消费而不是生产的债务，就是让你成为奴隶的陷阱（Sep 2, 2018）
