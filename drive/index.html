    Motivation Systems in a Few Games

    Pokémon

    In the image above and the following images:

    1. The arrow means one thing is benefiting from another thing, maybe in generating more or maybe in generating more efficiently.
    2. The rectangle means collectable or completable items. Most of time completing a task is a desirable thing. Sometimes meeting items that is rare to encounter can be a sur-prize which triggers dopamine. Most items can be used to decrease the difficulty in solving a problem, often obtained by users after completing a task.
    3. The circle means repeatable problem sets. The set can be map-level, room-level, enermy-level, skill-level (图、室、敌、技).
    4. The diamond means accumalatable currency, that can be exchanged to usable things that can satisfy curiosity / increasing efficiency / buy high rarity stuffs.
    5. The colored one means by doing or obtaining itself is sufficient enough to be desirable. By linking with colored ones, the transparent background shapes become desirable as proxies to desired ones.

    We use the Pokémon Red and Green as an example.

    1. The main path of walkthrough can be somehow linear. And not too many people replay the game for multiple times as in a roguelike game. And this path is the main progression the player cares as people consider beat the game as an achievement.
    2. Also along the way, completing the gym challenges to collect the badges are also rewarding.
    3. By beating the enermies, the player is rewarded by coins and xp. By themselves, these accumulated fake currency is not intrinsically desirable. It has to be a proxy to something that is desirable.
    4. The coins can be used to buy items, which will benefit in solving the problems without much cognitive load. This is like using a cheatsheet to lookup the words. And similar to fast progress in cheating to get a certificate. If the items in the game are visually appealing like the pokemons, they can be themselves desirable, but normally in life the items are like medicine pills, useful but not directly appealing as in restaurant foods.
    5. The characters in Pokemon are cute, imagining catching those monsters itself is already rewarding. And then we find out that the monsters can be useful in solving the problems we care, it becomes more desirable, like an Apple Macbook Air.

    Super Mario Bros.

    The pokemon is a typical RPG game where Mario is an action game.

    1. The first thing is that in Mario you can’t use coins or XPs to buy things to improve your ability in your future challegens. Only RPG-like stuff is that you can enter a new level with a super mario with an extra life, but it still like part of challenge instead of cheating.
    2. But the coins is not useless, it can give you extra life if you gather every 100 coins. That extra life give you more chance to progress more in path.
    3. Another benefit is that people want to compete with who has a higher score in one run.
    4. Unlike pokemon, the original super mario bros. often is played multiple times for most users as it is hard to progress too much and you have to start it over. So this game can be consider a Rogue-like game, only the levels is not changing much.
    5. Also unlike pokemon, your skills can be improve quickly by playing more mario, as in pokemon the improvement is more from levelup and better skill, there are less thing to master for normal players. This feature make mario more ACG.

    Duolingo

    Duolingo has game elements that seemingly borrowed from Candy Crush which was a phenomenal game at the launch of duolingo. And those tile-matching mobile games have similar features shared with Duolingo.

    1. Unlike in other games, the lexemes or tasks for the user to remember and master is often useless in real world. But in an education gamified app, this is a key motivation.
    2. Even though you can buy hearts to benefit your problem solving, which is like cheating in an exam, but this is a learning app not a testing app, mistakes should not be too punishing. But using hearts or energy, Duolingo can make this is a way to let serious users pay for a full game. So this feature is not a designed cheating as in an RPG.
    3. Duolingo’s main motivational forces are people’s desire to master a new skill. Its XP system is not so strong as it leads to highscores in leaderboard and unappealing quests daily. As we see in most console games, those are not major drives to motivate users and well game designed.

    怪物史莱姆

    1. 收集史莱姆，饲养史莱姆，收集果实，卖掉果实，都相当于使用技能是击打敌人
    2. 史莱姆产生的果实，相当于是金币奖励
    3. 升级牧场本省，是一些零散的关卡阶段
    4. 如果通关，一般意味着主要关卡都解锁

    最常见动力系统

    1. 如果整个游戏一遍过，那么推进主线就是核心目标
    2. 如果整个游戏反复玩，那么肉鸽反复体现就是目标
    3. 一般来说，都会有，图，室，敌，技，四个层级的题目系统
    4. 打击每个层级的题目后，会有xp或者coin的奖励
    5. 打击每个层级或者扫图中，会有道具或者技能的奖励
    6. 使用xp或者coin，一般可以增加做题的流利，可以提升道具或者技能的等级
    7. Boss虽然看起来像是一个敌人，但更像是 Level，像法环这种难度的，可以算是一个小型肉鸽游戏
    8. 有些游戏，有角色，例如，宝可梦，是特殊的一种道具，是技能的容器

    进一步展开

    1. 【进展】【Path】主线路径推进，消耗内容
    2. 【肉鸽】【Kill / Puzzle】反复遇到的短基本单元（15秒，短视频）
    3. 【肉鸽】【Battle / Level】反复遇到的长基本单元（3分钟，短的视频），会有精心的内容设计

    7. 【收集】特殊【道具】【Character】词汇的载体和容器，一个文件夹
    8. 【进展】【增滑】【Level Up】
