2023年高考数学

2023年全国甲卷压轴选择题

点 $P$ 在 焦点为 $F_1,\, F_2$ 的椭圆 $\frac{x^2}{9} + \frac{y^2}{6} = 1$ 上，且$\cos\angle F_1 P F_2 = \frac{3}{5}$，求 $OP$

  1. 当考生看到 📣 信号 椭圆 和 📣 信号 $PF_1$，$PF_2$时，应该能够列出 $PF_1 + PF_2 = 6$ 这样一个结果
  2. 我们可以类比成，玩家来到了 🗺 “椭圆的定义”这条道路通往的 🛏 房间，这个 🛏 房间里有一个 🎁 宝箱，宝箱给了玩家一个新的 🔑 钥匙 $PF_1 + PF_2 = 6$
  3. $\cos$ 这个 📣 信号，让考生想到了尝试进入余弦定理 🚏 通往的 🛏 房间
  4. 为了能打开装在 🛏 余弦定理通往的房间内的宝箱，考生找到了 $F_1F_2=2\sqrt{3}$ 这样一个 🔑 钥匙
  5. 打开了藏在 🛏 余弦定理通往的房间内的重要道具 🔑 $PF_1 \cdot PF_2 = 15/2 $
  6. 而为了 🔒 求解 $OP$，同样作为 📣 信号，让考生想尝试走中线定理这条 🚏 路径，去往相应房间
  7. 房间内有一个 🔒 上锁的宝箱，有着一个需要求解 $PF_1^ 2 + PF_2^2$ 的 🔑 钥匙要找到
  8. 而 $PF_1^ 2 + PF_2^2$ 意味着要 🔒 $PF_1 \cdot PF_2 $ 和 🔒 $PF_1 + PF_2 $ 这两个 🔑 宝箱钥匙要找到
  9. 而之前我们 🎒 背包里，有着这两把 🔑 钥匙，自然就解开了谜题，通过了 🏰 关卡

在这里，

  1. 考生无法像玩家一样，清晰的看到房间可能通往的路径，是考生凭借着经验，脑补出来的地图，在这个昏暗无光的 “王城下水道” 里探索
  2. 正向搜索： 📣 信号，刺激着考生沿着某条 🚏 路径 探索，路的尽头是一个 🛏 房间，房间内的挑战结束后会有一个 🎁 宝箱，宝箱会给考生一个 🔑 钥匙
  3. 反向搜索： 一个 🔒 上锁的宝箱需要找到 🔑 钥匙，而这个宝箱作为 📣 信号，指引考生沿着某条 🚏 路径 探索，路的尽头是一个 🛏 房间，房间内的挑战结束后会有一个 🔒 上锁的宝箱，宝箱作为 📣 信号，指引玩家去寻找对应的 🔑 钥匙

如果我们以 LLM 的角度来看这个问题：

  1. 那么考生要对题目的信息，进行解析 parse，找出对应的 📣 token
  2. 📣 token 作为输入，考生要找出这些 📣 token 所指引的几条高概率 🚏 处理路径 path 有那些
  3. 执行 🚏 处理路径，得到相应的新的 📣 信号，可能这个信号是 🔑 已知，也能是需要求解的 🔒 未知
  4. 直到挑战中要求的所有 🔒 上锁的宝箱都打开

那作为学习过程，我们需要训练学生的，就是：

  1. 对 📣 信号敏感熟悉，对其对应的几条高概率 🚏 处理路径有很好的模型
  2. 对 🚏 处理路径的处理过程执行熟悉，并熟练掌握
  3. 对解决问题普遍适用的 🚏 处理路径熟悉，并熟练掌握

2023年 新高考1 22

这个题选拔性太强了，属于高选拔性，不容易训练的技能

  1. 明显是要设两个未知数来表达周长
  2. 接下来都是高难度的操作，不容易训练
  3. 对于两个变量，想求最值，大概率要通过不等式消去一个变量
  4. 变成一个变量之后，就可以用导数技能来求最值
  5. 利用不等式降元是最困难的一个步骤

2023年 新高考2 17.1

  1. 为了利用面积条件，使用 $\frac{1}{2} b c \sin A$ 并不是明朗的路径
  2. 而另外一个路径：底乘以高除以2，更加容易得到新的信息，也就是 $a = 4$
  3. $BD$ 和 $AD$ 可以求出，就可以求出 $AB$ （$AC$ 也可以求出）
  4. 那么利用面积，或者余弦定理，都可以确定角 $B$

2023年 新高考2 17.2

  1. 看到 $b^2+c^2$，明确只要求出 $bc$，或者其他二者的方程就行
  2. 中点意味着要使用中线定理，得到 $a$
  3. 而为了得到 $bc$，意味着要求出 $A$
  4. 而 $b^2 + c^2$ 和 $A$ 的信号，都意味着使用余弦定理
  5. $bc$ 和 $A$，两个方程，两个未知数
  6. 得到 $bc$，从而得到 $b$ 和 $c$

2023年 新高考2 21

这个题目质量不高，有点无厘头，如果告诉定直线具体是什么，还好，在高考的紧张氛围下，这个题对思维和基本功的考察不是很成功。

  1. 因为 M，N 不能互换，暗示着韦达定理失效
  2. 过定直线，如果不是垂直于坐标轴的，难度系数极高
  3. 比较安全的办法就是，把M，N都解出来，硬算，或者都换成M的
  4. 另外就是，尝试看看这个定直线是否是特殊的，带入特值
  5. 猜想定直线就是 $x = -1$，然后就是赌一赌，搏一搏
  6. 根据结论逆着去证明，看需要证明什么结论这个问题可以解决

2023年 全国甲 11

这就是让高中生研究高斯函数的性质

  1. 求导发现函数在 $x=1$ 处前增后减，且恒大于零
  2. 增减性发现 $a < b$
  3. 而 c 的大小不容易判断，说明要研究函数的额外性质
  4. 奇偶性不是很明朗，但发现 $f(x+1)$ 是偶函数
  5. 对称性也不是一眼看出，只能猜测关于 $x=1$ 对称，然后证明
  6. 无论如何，通过对称，得到答案

2023年 全国乙 12

这个题目还是有难度的

方法一：代数方法

  1. 找到一个容易计算的原点，A 作为原点很容易计算
  2. 目标就是表示 D，而需要一个变量，显然 k 是usual suspect
  3. 用 k 来表示数量积，从而找到最值，类似圆锥曲线压轴大题的难度

方法二：几何方法

类似于 2010 年全国 I 理科数学的第 11 题：

  1. 设一个 $\theta$，用它表示要求的表达式
  2. 通过三角函数的技能，求出最值
  3. 难点在于，要想到设 $\theta = \angle OPD$
