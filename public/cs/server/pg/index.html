Commands

Backslash

  1. \l : list all dbs
  2. \c : connect to a db
  3. \du : list all user
  4. \d : list all tables
  5. \d <table_name>: list

Backup and restore

  1. pg_dump <database_name> > backup.sql
  2. pg_restore -d <database_name> backup.sql
