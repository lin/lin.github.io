VIM

  1. h / j / k / l: Left Down Up Right
  2. 0: line begin
  3. $: line end
  4. gg: file begin
  5. G: file end
  6. w / b: move word
  7. i: insert
  8. dd: delete current line
  9. :q: quit vim
  10. :q!: quite vim without saving
  11. :/: search
  12. :w: save
